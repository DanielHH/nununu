# Nununu backend

## Starting the server
  ```
  python server.py
  ```

## Testing
### Running tests
  Remember to list any new files that should be tested in .coveragerc

  ```
  coverage run server_test.py
  ```

### Coverage report
#### Create coverage report
  ```
  coverage html
  ```

#### View coverage report
Open the file htmlcov/index.html in a web browser
